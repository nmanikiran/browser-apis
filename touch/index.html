Multi-touch interaction

This app demonstrates using the Touch Events touch event types (touchstart, touchmove, touchcancel and touchend) for the following interaction:

  1. Single touch
  2. Two (simultaneous) touches
  3. More than two simultaneous touches
  4. 1-finger swipe
  5. 2-finger move/pinch/swipe
